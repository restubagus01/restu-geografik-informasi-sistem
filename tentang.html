Tentang Aplikasi

Web GIS Kecamatan merupakan aplikasi berbasis web yang dirancang untuk menyajikan informasi spasial dan non-spasial wilayah kecamatan secara interaktif. Aplikasi ini menampilkan peta kecamatan dan peta desa berbasis teknologi Sistem Informasi Geografis (SIG) dengan memanfaatkan Leaflet dan OpenStreetMap sebagai peta dasar. Web GIS ini bertujuan untuk mendukung penyajian informasi geografis, pembelajaran, serta analisis wilayah secara sederhana dan mudah diakses melalui media web.
